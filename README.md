# Trama Studio — Editorial Digital Portfolio

Digital interface designed for high-conversion and high-authority project presentation. Trama Studio focuses on "criterio" over generic execution, blending Swiss typographic principles with modern interactive storytelling.

## Visual Identity & Design System
- **Core Palette**: Deep Dark (#0B0A0C), Studio Orange (#D65027), Bone White (#F2EFE9).
- **Typography**: 
  - *Playfair Display*: Massive serif scales for editorial headers and statements.
  - *IBM Plex Mono*: Technical precision for labels, metadata, and secondary data.
- **Philosophy**: Minimalist-brutalist. Asymmetric grids (12-column), high-contrast hierarchy, and intentional micro-animations.

## Technical Foundation
- **Framework**: React + Vite for high-performance rendering.
- **Styling**: Tailwind CSS + Custom Design Tokens in `index.css`.
- **Motion**: Framer Motion / Custom Reveal components for scroll-based entry.
- **Architecture**: Modular component-based structure focusing on high-reusability and clean vertical spacing.

## Local Development

```bash
# Clone the repository
git clone https://github.com/nadiaescobbb/tramastudio.git

# Install dependencies
npm install

# Start development server
npm run dev
```
